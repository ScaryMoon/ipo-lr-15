# Лабораторная работа №15 “Разработка простого веб-сервиса” - Вариант 4
# Общие задания
### Задание 1
Создать простейший веб‑сервис на Flask.
Структура проекта:
task_1/
    app/
        __init__.py
        routes.py
    main.py
    README.md
В routes.py реализовать маршруты:
* / — приветствие
* /hello/<name> — персональное приветствие
* /square/'int:number' — квадрат числа
В main.py настроить запуск приложения.
